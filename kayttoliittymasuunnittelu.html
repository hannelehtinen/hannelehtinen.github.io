Valmis prototyyppi on rakenteeltaan melko samankaltainen kuin varsinainen tuote. Eroavaisuuksia prototyypissä on yksittäiset visuaaliset elementit, kuten ikonit, taustakuva ja varsinaiset tutoriaalinappulat. Lopullisen käyttöliittymän toteutin pääasiassa prototyyppiä mukaillen. Sovelluksen ulkoasu on toteutettu CSS:llä ja jonkin verran Angular Material -elementeillä, jotka olen itse muokannut sovelluksen tyyliin sopivaksi. Olin vastuussa sovelluksen visuaalisesta ilmeestä ja tyylittelystä kaikissa muissa komponenteissa paitsi quiz-komponentissa ja rekisteröitymis- ja kirjautumiskomponenteissa.



Tutoriaali


Alla tutoriaalin suoritustoiminto, johon olen tehnyt ulkoasun. Tutoriaali on kokonaisuudessaan responsiivinen. Tutoriaalin yksinkertaiset animaatiot on toteutettu animate.css -kirjastolla.



Alla CSS-koodi maskoteille eri tilanteissa. Eri elementit, kuten asusteet ja tassu on rakennettu erillisinä kuvina, jotka asetellaan päällekkäin. Päälle, tassulle ja päässä olevalle hatulle on määritetty sama wobble-animaatio.



    .correctSquirrel {
      margin-top: 5%;
      float: right;
      width: 50%;
  }
  
  .squirrel {
      position: relative;
      margin: auto;
      width: 200px;
      height: 200px;
  }
  
  .squirrelThumbsUp {
      position: relative;
      margin-top: -40px;
      margin-right: -10px;
      float: right;
      width: 200px;
      height: 200px;
  }
  
  .body, .face, .accessory, .book, .paws, .thumb, .thumbsUp, .thumbsUpAccessory {
      position: absolute;
      width: 200px;
      height: 200px;
  }
  .face, .thumb, .accessory {
      z-index: +1;
      animation: wobble 1s ease-in-out infinite;
      animation-direction: alternate;
  }
  
    @keyframes wobble {
      from {
          transform: rotateZ(0deg);
      }
      to {
          transform: rotateZ(3deg);
      }
    }


Kauppa ja Inventaario


Kauppa ja Inventaario -ikkuna on rakennettu CSS Grid:in avulla. Grid oli minulle ennestään pääpiirteittäin tuttu, mutta en kokemusta siitä ei pahemmin ollut. Toteutin asettelun ensin Angular Material Gridin avulla, mutta vaihdoin sen tavalliseen Gridiin saadakseni helpomman mahdollisuuden kustomointiin ja responsiivisuuteen. Inventaarion Equip-nappi muuttuu sen mukaan, kellä maskotilla on/ei ole esine käytössä.




Pop Up -ikkunat


Pop up -ikkunat on tehty Angular Material -komponenteilla. Tein tyylittelyt ja kuvitukset sekä asettelun ikkunoihin. Koska elementit on tehty Angular material -dialogia käyttäen, tyylittely vaati enemmän olemassaolevien tyylien yliajamista.



Responsiivisuus



Sovellus on suunniteltu alunperin mobiilille ja responsiivisuuden olen lisännyt jälkeenpäin. Olennaisin muutos nettisivuversiossa on alapalkin siirtyminen yläpalkiksi, sekä erillinen ikkuna, jossa näkyy käyttäjän tärkeimmät tiedot.




Animaatio



Täysin uutena asiana itselleni opettelin käyttämään Angularin Animaatiokomponenttia. Tein sovellukseen pienen Router Animaation, jolloin sivua vaihtaessa sisällöllä on Fade In -animaatio. Alla Typescript-koodi animaatiosta. 



      import {
        trigger, transition, style, query, animate
      } from '@angular/animations';
    
    export const fader =
      trigger('routeAnimations', [
        transition('* <=> *', [
          query(':enter, :leave', [
            style({
              position: 'absolute',
              left: 0,
              width: '100%',
              opacity: 0,
            }),
          ], { optional: true }),
          query(':enter', [
            animate('1500ms ease', style({ opacity: 1 })),
          ], { optional: true })
        ]),
    ]);
